# altkey.tcl

`altkey.tcl` is a command line tool for for calculating keyboard
accelerators (e.g., for menu options or dialog labels).

See `input.txt` for a test input file and `expected.txt` to see the output.

Note: I use [Store](https://github.com/mark-summerfield/store) for version
control so github is only used to make the code public.

## Dependencies

Tcl/Tk >= 9.0.2.
